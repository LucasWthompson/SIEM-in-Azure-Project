<h1>SIEM in Azure</h1>


<h2>Description</h2>
Project consists of using applications in Azure, such as Virtual Machines, Sentinel, and Log Analytics Workspace to create a SIEM in Azure. 
<br />


<h2>Utilities Used</h2>

- <b>Microsoft Azure</b> 
- <b>Virtual Machines in Azure</b>
- <b>Microsoft Sentinel in Azure</b>
- <b>Log Analytics workspaces in Azure</b>
- <b>PowerShell</b>
- <b>ipgeolocation</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
